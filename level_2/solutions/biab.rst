Beacon-in-a-Box (BiaB) for Level 2 onboarding
------------

This solution for a Level 2 connection is based on GA4GH Beacon v2 Reference Implementation (B2RI) reference software [`link <https://b2ri-documentation.readthedocs.io/en/latest/beacon-v2-reference-implementation/>`_], a **non-dockerised** version of BiaB is developed for our customised EJP RD Beacon v2 API Specification.
